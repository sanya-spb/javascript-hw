Практическое задание к Уроку #2

  1. Почему код дает именно такие результаты?

    var a = 1, b = 1, c, d;
c = ++a; alert(c);           // 2
d = b++; alert(d);           // 1
c = (2+ ++a); alert(c);      // 5
d = (2+ b++); alert(d);      // 4
alert(a);                    // 3
alert(b);                    // 3

    Ответ: Префиксная форма возвращает новое значение, в то время как постфиксная форма возвращает старое (до увеличения/уменьшения числа).

  2. Чему будет равен x?

    var a = 2;
var x = 1 + (a *= 2);

    Ответ: перепишем математически: x = 1 + (a * 2) = 1+(2*2) = 5
